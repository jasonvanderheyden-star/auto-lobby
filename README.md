# Auto Lobby

Automated Canadian lobbyist compliance — federal first (OCL), provincial + municipal later. Multi-tenant B2B SaaS. Deep Sky is the first user.

## Getting started with Claude Code

This repo is built collaboratively with [Claude Code](https://claude.com/claude-code). Install it once:

```bash
npm install -g @anthropic-ai/claude-code
```

Then from this directory:

```bash
claude
```

Kick off the first session with:

> Read `CLAUDE.md` and the linked `docs/` files. Then execute Phase 0 task 1 (bootstrap Next.js app with App Router + TS + Tailwind + ESLint, set up Prettier). Commit when ready and pause for my review before continuing.

Claude Code will read the brief, scaffold the project, and stop for you to review before moving on.
